BOTH-POOL DEFINITION — BILINGUAL CITY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Valencia is bilingual (ES / Valencian CA) with an English expat and
travel writing layer. Both-pool requires:
- Two editorially independent sources
- From different publishers
- Writing in different languages

VALID both-pool combinations:
- EN source + ES source (different publishers) ✓
- EN source + Valencian CA source (different publishers) ✓
- ES source + Valencian CA source (different publishers) ✓

NOT valid:
- Ojo al Plato article 1 + Ojo al Plato article 2: same publisher ✗
- Any two articles from the same publication ✗

VALENCIAN-LANGUAGE NOTE: The v6 pack had no Valencian/CA-language
source. Valencian is spoken widely in Valencia and some food media
publishes in Valencian. Finding a named-author Valencian-language
source in Phase 1 would open a new both-pool lane. Prioritise
Valencian-language discovery in Phase 1 searches.

regional guides frequently include restaurants from across Comunitat
Valenciana — reject any entry not confirmed as being within Valencia
city limits.

GIT HISTORY NOTE — REGRESSION THRESHOLD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The v6 Valencia pack (47R, 10BP) was built from 7 named-author sources,
all of which pass current quality gates. However, some of the v6
both-pool entries (Ricard Camarena, El Poblet) were Michelin-starred
restaurants now subject to blanket exclusion. Expect some regression
on that dimension — this is correct.

A significant both-pool regression beyond the Michelin exclusions would
indicate a pipeline search failure. Minimum acceptable for a successful
v7.1 run: 5+ both-pool entries (accounting for 2-3 Michelin exclusions
from the v6 both-pool). If both-pool falls below 4, check the Phase 0
fetch log before accepting.
