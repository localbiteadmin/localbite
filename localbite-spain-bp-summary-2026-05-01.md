## Key Findings

### 1. Definition change is the primary lever
The definition change from language-diversity to publisher-independence delivers +73 BP (+10.9pp) across the Spain fleet with no pipeline work — just a recomputation script. New sources discovered add at most +3–5 BP on top of that, and only for Barcelona and Madrid.

### 2. Language-gap cities — definition change is the sole fix
Seven cities have all sources in a single language. No new sources were found through Phase C searches. The definition change unlocks them entirely:

| City | Current | After def change | Gain |
|------|---------|-----------------|------|
| A Coruña | 0% | 44% | +44pp |
| San Sebastián | 8% | 38% | +30pp |
| Santiago | 11% | 30% | +19pp |
| Murcia | 0% | 30% | +30pp |
| Vigo | 0% | 20% | +20pp |
| Zaragoza | 0% | 13% | +13pp |
| Valladolid | 0% | 13% | +13pp |

### 3. Seville is structurally broken — no fix available
All 22 restaurants are single-source. No new source was found. Definition change has no effect. Seville needs a full editorial rebuild with sources covering overlapping restaurants — not a definitional fix.

### 4. Vocento group conflict — fleet-wide constraint
Grupo Vocento owns: ABC, El Diario Vasco (San Sebastián), El Correo (Bilbao), Las Provincias (Valencia), Ideal (Granada), Sur (Málaga). Any city where Maribona/ABC is added cannot also use these Vocento papers as an independent source for the same city. Currently only affects Madrid (Maribona confirmed). Flag before expanding Maribona to other cities.

### 5. Phase C pattern across Spain
Restaurant-level searches for smaller Spanish cities (Córdoba, Logroño, Pamplona, Zaragoza, Santiago, Bilbao, Valladolid, A Coruña, Murcia, Vigo, Palma, Alicante, Seville) returned only aggregators and sources already in the pack. This confirms the Spain writer discovery gap is a structural one, not solvable by more search — it requires knowing specific writer names in advance (pre-research approach).
