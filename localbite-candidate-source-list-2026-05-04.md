## Critical Ownership Maps — Read Before Any Verification Run

### Italy — Four Group Map

| Group | Owner | Fleet-city papers |
|-------|-------|------------------|
| **GEDI** (being sold to Antenna/Greece) | Agnelli/Exor → Antenna | La Stampa (Turin), Il Secolo XIX (Genoa), and via NEM subsidiary: Il Piccolo (Trieste), Messaggero Veneto (Friuli), La Nuova Venezia |
| **RCS** (Cairo Communication) | Urbano Cairo | Corriere della Sera (Milan), Corriere Fiorentino (Florence), Corriere del Veneto (Venice), Corriere di Bologna (Bologna), Corriere del Mezzogiorno (Naples) |
| **Caltagirone** | Francesco Caltagirone | Il Messaggero (Rome), Il Mattino (Naples), Il Gazzettino (Venice), Quotidiano di Puglia / Nuovo Quotidiano di Puglia (Lecce/Bari) |
| **Monrif / Editoriale Nazionale** | Monti Riffeser family | La Nazione (Florence), Il Resto del Carlino (Bologna), Il Giorno (Milan), Gazzetta di Modena |

**Independence rule:** Two papers from the same group cannot form a consensus pair **within the same city's pack.** Papers from the same group in *different* cities are fine — each city pack is evaluated independently.

**Key intra-city conflicts to know:**
- If **CdS/Visintin (RCS)** is added to Milan → Corriere di Bologna (RCS) *cannot* pair with it in Milan (irrelevant, different cities, no conflict in practice)
- If **La Stampa (GEDI)** is added to Turin → Il Secolo XIX (GEDI) *cannot* pair with it in Turin (also irrelevant, Secolo is in Genoa)
- **Caltagirone** owns papers in both Naples (Il Mattino) and Venice (Il Gazzettino) — they can each be in their respective city packs independently. But within Naples, Il Mattino cannot pair with lucianopignataro.it *only if* that site is Caltagirone-owned (it is not — it's independent). Il Mattino can pair with other independent sources in Naples.
- **HOWEVER:** Il Mattino shares editorial voice with lucianopignataro.it (Luciano Pignataro is the Naples food critic *at* Il Mattino). This is not an ownership conflict but a **same-author conflict** — adding Il Mattino to Naples would add the same human editorial voice already in the pack via his personal site. Confirmed in previous session: Naples / Il Mattino excluded on same-editorial-voice grounds, not ownership.

**Critical implication:** Vocento's ABC (Maribona) is being added to Madrid. Vocento papers cover 7 other fleet cities. **Within each of those cities**, if the only regional paper is a Vocento paper, that Vocento paper can still be added as a source — it's independent from *non-Vocento* sources already in those packs. The conflict only arises if you try to pair two Vocento papers **within the same city**. Since Madrid's pack gets Maribona/ABC and Madrid's regional press is national, there's no cross-city conflict.

**The real Vocento problem:** For Bilbao, San Sebastián, Logroño, Valladolid, Murcia, Granada, Málaga — if the dominant local paper is Vocento, and we want a named critic from that paper, we need to check whether any other Vocento source is *already in that city's pack*. If none are, the Vocento paper qualifies. If one is, the Vocento paper cannot pair with it.
